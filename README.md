# llmon-:pie:

local cli/gui llm chat with speech-to-text and text-to-speech support

uses llama-cpp-python, coqui.ai/xtts, pywhispercpp, and streamlit

to use:

look at code, modify it. 

streamlit run gui-llmon.py
